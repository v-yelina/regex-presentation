RegExp.prototype.test()


			const sea = /sea/;
			sea.test('I want to see the sea!');  // returns true
			/sea/.test('I want to see the sea!');  // returns true

Global flag


			const globalSea = new RegExp('sea*', 'g');
			globalSea.lastIndex // returns 0
			globalSea.test('I want to see the sea!');  // returns true
			globalSea.lastIndex // returns 12
			globalSea.test('I want to see the sea!');  // returns false

Writing a regular expression pattern

/abc/

/ab*c/

Special characters categories

  • Assertions: ^, $, x(?=y), x(?!y), (?<=y)x, (?
  • Character classes: \d, \D, \f, \n, \r, \s, \S, \t, \v, \w, \W
  • Groups and backreferences: (x), (?:x), (?x), x|y, [xyz]
  • Quantifiers *, +, ?, x{n}, x{n,}, x{n,m}
  • Unicode property escapes: \p{UnicodeProperty}
